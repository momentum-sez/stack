# Open Banking / Account-to-Account Corridor Corridor Playbook (Reference)

This playbook describes the **institutional + technical** steps to activate the corridor.

## 1) Legal agreements

- Corridor participation memorandum (between zone authorities or designated operators)
- Data sharing agreement / DPIA alignment
- Dispute resolution and supervisory cooperation protocol

## 2) Compliance eligibility

- Verify each participant can produce all required attestations:
  - KYC/KYB + BO
  - sanctions screening
  - AML program controls
  - license validity
- Establish trust anchors (issuer allowlists) and key rotation policies.

## 3) Technical onboarding

- Configure payment/banking adapter endpoints
- Configure message formats and reconciliation
- Test end-to-end settlement scenarios (happy path + reversals + exceptions)

## 4) Operations & monitoring

- Define SLAs (availability, incident response)
- Define reporting cadence and metrics
- Define de-risking playbook for correspondent exit
