# Akoma Ntoso templates module

This module provides **starter templates** for:

- Acts (primary legislation)
- Regulations (secondary legislation)
- Bylaws / Rulebooks (authority-issued rules)

## How to use

Copy the relevant template and then edit:

- metadata: FRBR identifiers, dates, authors
- structure: sections, definitions, schedules
- stable `eId` anchors for every clause

These templates include recommended `eId` naming conventions and placeholder values.
