# Crypto-Asset Service Provider License Module

This module provides a **complete licensing scaffold**:

- application form schema + example payload
- OPA/Rego eligibility and ongoing compliance checks
- periodic reporting return schema + example
- incident report schema + example
- a policy-to-code map describing required attestations and regulator views

> This is a **model** scaffold. Deployments must align with local law and supervisory practices.

## Artifacts

- `forms/application.schema.json`
- `policy/eligibility.rego`
- `policy/ongoing.rego`
- `reporting/periodic-return.schema.json`
- `reporting/incident-report.schema.json`
- `src/policy-to-code/map.yaml`
